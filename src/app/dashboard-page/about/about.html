Web Drone GCS

    Web Based Ground Control Station for Drones    

WebGCS provides full flight control and configuration for ArduPilot or PX4 Pro powered vehicles (any device who is using MavLink protocol (v1,v2)).
The goal for WebGCS is improved ease of use for new users as well as high end feature support for experienced users.
Amount the GCS you can find the many features and capabilities such as the below list
•
Full configuration support for vehicles running PX4 Pro and ArduPilot (ArduCopter, ArduPlane) firmware
Mission planning for autonomous flight
Flight map display showing vehicle position, flight track, waypoints and vehicle instruments
Video streaming with instrument display overlays
Flight support for any MAVLink capable vehicle

Requirements
•
Any supported OS below
Any Web Browser
JAVA 1.8

Web GCS can be installed in several different setups and methodology, a multi-deploy or a standalone.
Each have is own benefits, it depand on the use case, equipment and flight conditions.

•

Deloyemng a standalone method a simple one, it means all the information (VTX, RX/TX) are being send and received from a single source machine With a dedicated idea

•

Deloyemng a multipath method is fairly simple, safer and more robost. under this deployment the main server is responsible of telemtry/radio part.
Including the mission planning and many more. the video input (VTX) is being handle by external devices.

Supported Platforms



The Ground Control Station can work on varied amount of operation system (as mentioned above).
the GCS have a backend communicator which can be executed on any platform support JAVA Language and JVM (Mobile devices are don't support such cases).
If you want to use mobile device than you can always set an external device with the requirement above and connect to it remotely.
